Into the future

Or, what are the critical next steps to improve these offerings?

Easy, out-of-the-box API standards, layouts and documentation

  • The methods and descriptions for DOI APIs are all over the map
  • Working on providing suggestions and templates for creators so we can start to converge
  • Great discussions going on about 'what makes a great API'

A more functional datahub that pulls it all together

  • Next-gen /data
  • Provide easy access to all data (APIs and other)

Citizen contributions to official datasets

  • Would love to see an engraned workflow to let citizens improve datasets
  • This would be extemely valuable in particular for public lands (trail mapping, invasive species, etc)
